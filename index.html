¡Bienvenido a EscolApp!
Configura tu app en 4 pasos sencillos.
Cada familia gestiona sus propios datos de forma privada.
1
Crear proyecto Firebase
Firebase es la base de datos gratuita de Google que permite colaboración en tiempo real entre padres e hijos. Cada familia tiene su propio proyecto — tus datos son completamente privados.
🔗 Abrir Firebase Console ↗
En Firebase Console:
1
Crear proyecto
Toca "Crear un proyecto" → nombre: EscolApp → desactiva Analytics → Crear.
2
Agregar app web
Menú ⚙️ → Configuración → sección "Tus apps" → ícono </> → nombre: EscolApp → Registrar.
3
Copiar credenciales
Copia el objeto firebaseConfig que aparece. Lo necesitas en el paso 2.
4
Activar Authentication
Menú lateral → Authentication → Comenzar → pestaña "Sign-in method" → Google → Activar.
5
Activar Firestore
Menú lateral → Firestore Database → Crear base de datos → Modo producción → región us-east1.
2
Pegar credenciales Firebase
Pega aquí el objeto firebaseConfig que copiaste. Tiene este formato:
const firebaseConfig = {
  apiKey: "AIza...",
  authDomain: "escolapp-xxx.firebaseapp.com",
  projectId: "escolapp-xxx",
  storageBucket: "escolapp-xxx.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456789:web:abc..."
};
3
Gemini API (clasificación IA)
Gemini analiza automáticamente el asunto de cada correo y sugiere categoría, asignatura y fecha. Es 100% gratuito hasta 1,500 solicitudes/día.
🔗 Obtener API Key gratis ↗
En Google AI Studio:
1. Inicia sesión con tu cuenta Google
2. Toca "Create API Key"
3. Copia la clave y pégala aquí:
Si no la configuras ahora, la app usará clasificación por palabras clave.
4
Tu perfil en EscolApp
¿Cómo quieres aparecer en los tableros compartidos?
💡 Sobre los roles en los tableros
Admin: crea tableros, invita miembros y cambia roles.
Colaborador: crea y edita tarjetas.
Lector: solo ve y marca tareas como listas.

Puedes tener múltiples admins por tablero. Un padre puede ser admin en el tablero de un hijo pequeño y solo colaborador en el de un hijo mayor.
